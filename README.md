## 🚀 Features

- **Cinematic Aesthetics**: Premium design featuring a dynamic sunset/sunrise environment.
- **Glassmorphism**: Sleek frosted glass panels for the authentication form.
- **Micro-Animations**: Interactive hover effects and smooth transitions that bring the UI to life.
- **Responsive Layout**: Seamlessly adapts from desktop monitors to mobile devices.
- **Performance Optimized**: Built with pure HTML, CSS, and Vanilla JavaScript. No heavy frameworks.

## 🏃‍♂️ Getting Started

To run this project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/Prajjaval08/LogIn-Page--frontend.git
   ```
2. Navigate into the directory
   ```bash
   cd LogIn-Page--frontend
   ```
3. Open `index.html` in your favorite browser or start a local server.
